# Aya-slman
تعريف حجم المخزن

تم اختيار حجم صغير للبافر من أجل إظهار خاصية الالتفاف بوضوح:

#define SIZE 5

---

تركيب المخزن الدائري

تم تعريف هيكل بيانات يحتوي على العناصر الأساسية التالية:

typedef struct {
    char buffer[SIZE];
    int head;
    int tail;
    int count;
} CircularBuffer;

شرح العناصر:

- buffer: مصفوفة لتخزين البيانات.
- head: مؤشر القراءة.
- tail: مؤشر الكتابة.
- count: عدد العناصر الحالية داخل البافر.

---

شرح الدوال المستخدمة

دالة التهيئة

void init(CircularBuffer *cb)

تقوم بتهيئة المخزن الدائري إلى الحالة الابتدائية من خلال:

- ضبط مؤشر القراءة إلى 0
- ضبط مؤشر الكتابة إلى 0
- تصفير عدد العناصر

---

دالة التحقق من الامتلاء

bool isFull(CircularBuffer *cb)

تعيد قيمة صحيحة إذا كان البافر ممتلئًا بالكامل.

---

دالة التحقق من الفراغ

bool isEmpty(CircularBuffer *cb)

تعيد قيمة صحيحة إذا كان البافر فارغًا.

---

دالة الكتابة داخل البافر

void writeBuffer(CircularBuffer *cb, char data)

تقوم بإضافة عنصر جديد داخل البافر عبر:

1. التحقق من وجود مساحة فارغة.
2. تخزين العنصر في موقع الكتابة.
3. تحريك مؤشر الكتابة للأمام باستخدام الالتفاف.

---

دالة القراءة من البافر

char readBuffer(CircularBuffer *cb)

تقوم بقراءة عنصر من البافر عبر:

1. التحقق أن البافر غير فارغ.
2. قراءة العنصر من موقع القراءة.
3. تحريك مؤشر القراءة للأمام.

---

آلية الالتفاف

يعتمد البرنامج على المعادلة التالية لتحقيق الالتفاف:

(index + 1) % SIZE

الفائدة:

عند وصول المؤشر إلى آخر عنصر في المصفوفة:

- يعود تلقائيًا إلى أول عنصر.

---

معالجة الحالات الخاصة

امتلاء المخزن (Overflow)

عند محاولة الكتابة داخل بافر ممتلئ:

- يتم منع الإدخال.
- تظهر رسالة تنبيه.

---

فراغ المخزن (Underflow)

عند محاولة القراءة من بافر فارغ:

- يتم منع القراءة.
- يتم إرجاع قيمة فارغة
